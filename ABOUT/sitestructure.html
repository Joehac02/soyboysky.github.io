SITE STRUCTURE

I am currently devising official site structure templates for the different types of documents on my site. Currently, there are an overarching 5 different types:

  1. Home page - the index, slight differences from link pages.
  2. Link page - a transition page that links to different sub-pages. (ex, notebook.html)
  3. Essay Page - One header + sub-header, and text. (ex, notebook.html)
  4. Table page - A page with sub-headers, text, and tables/lists.(ex, morningroutine.html)
  5. Link+ page - a page with main links and main characteristics of other pages. (ex, works.html)
